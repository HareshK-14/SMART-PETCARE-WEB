# Smart Pet Care Platform

Welcome to the root repository of the Smart Pet Care Platform. 

## Project Structure
- `database/` - Contains the PostgreSQL database schema and seed data.
- `backend/` - (To be generated) The Spring Boot REST API application.
- `frontend/` - (To be generated) The React web application.
- `docs/` - Project documentation.

## Running the Database
To initialize the backend requirements, run the SQL script located at `database/schema.sql` inside your local pgAdmin or PostgreSQL console.
